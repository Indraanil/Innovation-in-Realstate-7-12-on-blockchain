## ðŸ’¡ Judge Q&A Preparation

### Technical Questions

**Q: Why Algorand over Ethereum?**
A: Algorand offers:
- Lower transaction fees (crucial for Indian market)
- Faster finality (4.5 seconds)
- Carbon-negative (important for ESG)
- Built-in asset creation (ASA)
- Better for high-frequency trading

**Q: How do you handle document verification?**
A: Multi-layered approach:
1. OCR extraction (Tesseract + OpenCV)
2. ML fraud detection (Random Forest classifier)
3. Image quality analysis
4. Pattern recognition
5. Anomaly detection (Isolation Forest)

**Q: What about legal compliance?**
A: 
- KYC/AML verification (Aadhaar + PAN)
- DigiLocker integration for government docs
- Land registry linking
- Tax reporting (capital gains, rental income)
- Clear disclaimer: blockchain is verification layer, not legal replacement

**Q: How do smart contracts work?**
A: PyTeal contracts on Algorand handle:
- Token minting (ASA creation)
- Ownership tracking
- Automated income distribution
- Governance voting
- Escrow for trades
